# SynthoCore Brand Assets

This directory contains official SynthoCore brand assets.

## Logo

The official SynthoCore logo is hosted at:

```
https://iili.io/Cor8wbe.md.jpg
```

### Usage in Markdown

```markdown
<img src="https://iili.io/Cor8wbe.md.jpg" alt="SynthoCore" width="120" />
```

### Available Files

| File | Description |
|---|---|
| `logo.png` | Primary logo (dark background) |
| `logo-white.png` | White variant (dark backgrounds) |
| `logo-dark.png` | Dark variant (light backgrounds) |
| `logo.svg` | Vector source (preferred for print) |
| `icon.png` | Square icon (512×512, for favicons) |
| `banner.png` | GitHub social preview banner (1280×640) |

## Color Palette

| Name | Hex | Usage |
|---|---|---|
| **Synco Blue** | `#0052FF` | Primary brand color (Coinbase CDP blue) |
| **Synco Cyan** | `#00C2A8` | Accent color |
| **Synco Black** | `#0A0A0A` | Background |
| **Synco White** | `#FAFAFA` | Text on dark backgrounds |
| **Synco Gray** | `#8A8A8A` | Secondary text |

## Typography

- **Headings:** Inter (700, 800)
- **Body:** Inter (400, 500)
- **Code / Monospace:** JetBrains Mono

## Usage Guidelines

- Always maintain clear space equal to the height of the "S" in SynthoCore around the logo
- Do not alter colors, proportions, or add effects to the logo
- Use the white variant on dark backgrounds and dark variant on light backgrounds
- The logo may be scaled proportionally but never stretched
